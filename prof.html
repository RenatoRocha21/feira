BONNY KATHY SOARES DOS SANTOS

Matéria Descrição
Programação Orientada a Objetos I A programação orientada a objetos (POO) é um paradigma de programação que se baseia no conceito de "objetos", que são instâncias de classes. Uma classe é um modelo que define características e comportamentos comuns a um conjunto de objetos relacionados. A POO organiza o código em unidades autônomas, chamadas de objetos, que interagem entre si por meio de métodos, propriedades e eventos.

Matéria Descrição
Lógica e Linguagem de Programação A linguagem de programação é um conjunto de regras e instruções utilizadas para criar programas de computador. Ela é uma forma padronizada de comunicação entre um ser humano e um computador, permitindo que o programador especifique as ações que o computador deve executar. As linguagens de programação podem variar em níveis de abstração, indo desde linguagens de baixo nível, que são mais próximas da linguagem de máquina, até linguagens de alto nível, que são mais próximas da linguagem humana.
